# Engineering Foundations

This repository contains core software engineering concepts that are independent of any specific language, framework, or tech stack.

Focus is on understanding how systems actually work, not just how to use tools.

---

## Scope

Covers fundamentals used across all domains:

- Version Control
- Operating Systems
- Networking
- Databases
- APIs
- System Design
- Software Design Principles
- Testing
- Debugging
- Security Basics
- CI/CD
- Containers & Environments
- Cloud Basics

---

## Structure

Each folder represents a core domain.  
Inside each folder:

- `README.md` в†’ overview + key concepts
- topic files в†’ deeper explanations

---

## Why This Exists

Framework knowledge expires.  
Fundamentals compound.

This repo is a long-term reference for:

- understanding underlying systems
- improving problem-solving
- writing better, maintainable code

---

## Usage

- Read linearly or jump by topic
- Treat as notes, not tutorials
- Expand as new concepts are learned
